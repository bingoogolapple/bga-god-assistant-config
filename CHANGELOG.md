# CHANGELOG

## 0.2.0 - 2025-04-12

## 0.1.9 - 2024-11-17

- 支持在低代码区块模板市场中预览区块模板
- 添加几个工具方法
  - $utils 添加 fixAntFormilyItemMarginBottom 工具方法，快速修复区块样式问题
  - $utils 添加 setAsBlockRoot 工具方法，标记指定节点为远程动态区块根节点，方便子节点查找
  - $utils 添加 getBlockRootField 工具方法，方便快速获取区块根节点
